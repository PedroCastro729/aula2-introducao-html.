DOCTYPE
Indica o tipo do documento para o navegador
html
Onde fica todos os itens da página
head
Onde fica tudo que não é conteúdo (ajustes, programação, etc)
body
Onde fica o que aparece na página
h1
Para marcar um título em HTML
style
Insere um estilo do css
p
Para marcar um parágrafo em HTML
script
insere instruções do javascript
img
Adiciona uma imagem
br
Insere uma quebra de linha
ul
Insere uma lista não ordenada
ol
Insere uma lista ordenada
li
Marca um item da lista
dl
Insere uma lista de definição
dt
Marca o termo a ser descrito
dd
Marca a explicação do termo
